What I've Learned in this Class!

This class taught me much more than I would have initially thought. I walked into the class thinking that I would have learned only about how to program a game but I then realized that there is much more to game deveopment than just coding. I learned that I had to take into consideration what the market wanted, which game engines to even consider based on many factors of my desired game (ex: 2D games sometimes require different game engines than 3D games and different game engines utilize different programming languages), how to monetize my potential game, the competition that exists within the genre of my game, which tools and services to use when publishing and maintaining my game, and much more! I learned about various game engines like Unity, Gamefroot, and GameSalad and even learned how to hardcode my own game with only a JavaScript compiler! I will share with you two games that I've made: one that uses Gamefroot and the other that only uses a JavaScript compiler!
